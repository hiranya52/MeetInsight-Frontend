# 🚀 MeetInsight Frontend

![React](https://img.shields.io/badge/React-Frontend-blue)
![Angular](https://img.shields.io/badge/Angular-Frontend-red)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-Styling-38BDF8)
![License](https://img.shields.io/badge/License-MIT-green)

**MeetInsight Frontend** is a modern and minimal UI for the **Meeting & Interview Feedback Management System**.  
It provides dashboards, structured feedback forms, meeting management, and analytics for teams.

---

# ✨ Features

### 📊 Dashboard
- Meeting statistics
- Upcoming interviews
- Pending feedback
- Analytics cards

### 📅 Meeting Management
- Create meetings
- View meeting details
- Manage participants
- Track meeting history

### ⭐ Feedback System
- Structured interview feedback forms
- Rating system
- Strengths & weaknesses notes
- Comments & discussion

### ✅ Action Items
- Create tasks after meetings
- Assign task owners
- Set deadlines
- Track progress

### 📈 Analytics
- Candidate score comparison
- Interview rating trends
- Performance insights

# ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/MeetInsight-Frontend.git
```

### 2️⃣ Navigate into the project

```bash
cd MeetInsight-Frontend
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm start
```

Application will run on:

```
http://localhost:3000
```

or

```
http://localhost:4200
```

# 🔗 Backend Integration
